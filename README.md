# Polymarket Polygon 一键全仓赎回

## 功能
- 浏览器钱包连接
- 自动扫描已 Resolve 仓位
- 一键批量赎回
- Vercel 一键部署

## 一键部署

部署前请先上传本仓库到 GitHub，然后访问：

https://vercel.com/new/clone?repository-url=https://github.com/你的用户名/polymarket-redeem-vercel

## 本地运行

```bash
npm install
npm run dev
```

## 安全
- 本项目不接触私钥
- 所有交易通过钱包签名
- 建议先用小钱包测试
